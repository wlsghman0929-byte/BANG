# 📈 Quant Dashboard — 한국·미국 주식 퀀트 툴킷

두 가지 프로그램이 들어 있습니다.

1. **🛰️ 시장 스캐너 (`run_scanner.py`)** — 야후(yfinance)로 러셀1000 등 미국 종목을
   대량으로 끌어와, **급등·급락·거래량 급증·변동성 상위·기술적 추천**을 자동으로 뽑고
   일정 주기마다 갱신합니다. (이번에 새로 추가, 권장)
2. **📈 백테스트 대시보드 (`run.py`)** — 지정 종목(한국 KRX + 미국)에 대해
   **스크리너 · 백테스트 · 매매신호 · 자산곡선 시각화**를 돌립니다.

---

## ⭐ 시장 스캐너 (자동 갱신) — `run_scanner.py`

```bash
pip install -r requirements.txt
python run_scanner.py
```

- 실행하면 `scan_dashboard.html` 이 생성됩니다. **브라우저로 열어두면** 설정한 초마다
  자동 새로고침되고, 백그라운드의 파이썬이 N분마다 다시 스캔해 내용을 갱신합니다.
- **Ctrl+C** 로 종료합니다.
- 상단의 **🇺🇸 미국 / 🇰🇷 한국 토글**로 시장을 전환합니다. (미국=yfinance, 한국=pykrx)
- 탭 구성: ⭐기술적 추천(BUY) · 📈급등 TOP · 📉급락 TOP · 🔊거래량 급증 · ⚡변동성 상위.
  각 행에 티커+종목명·현재가·등락·거래량배수·ATR변동성·RSI·추천점수·신호·30일 스파크라인.
- **종목 행을 클릭**하면 가격 차트 + 매수세/매도세(거래량 색상 + 누적 매수세) 팝업이 열립니다.
- **예측마감** 컬럼: 추세(모멘텀 EWMA) + 평균회귀(RSI) + 변동성을 섞어 **다음 장 마감가**를
  추정합니다. 표에는 예측가와 예상 등락%, 모달에는 예측 범위(±1σ)까지 표시됩니다. 참고용 추정치입니다.

### 스캐너 설정 — `scan_config.py`

```python
MARKETS = ["US", "KR"]          # 스캔할 시장: ["US"], ["KR"], 또는 둘 다
US_UNIVERSE = "russell1000"     # "sp500" | "sp500+nasdaq100" | "russell1000" | "custom"
KR_UNIVERSE = "kospi200"        # "kospi200" | "kospi" | "kosdaq" | "kospi+kosdaq" | "custom"
MAX_TICKERS = None              # 속도 위해 시장별 상한 (예: 300)
REFRESH_MINUTES = 10            # 몇 분마다 재스캔 (실제 데이터 갱신)
AUTO_RELOAD_SECONDS = 20        # 브라우저 화면 새로고침 주기 (짧을수록 자주)
TOP_N = 30                      # 각 탭 상위 표시 개수
BIG_MOVE_PCT = 5.0              # 이 % 이상 등락이면 '급변동' 강조
VOL_SURGE_MULT = 2.0            # 평균 거래량 이 배수 이상이면 '거래량↑' 강조
RSI_BUY_MIN, RSI_BUY_MAX = 45, 68   # 추천 RSI 적정 범위
```

> 미국 러셀1000(~1000종목)·한국 코스피200(~200종목) 첫 스캔은 시세 다운로드에
> 수 분 걸릴 수 있습니다. 빠르게 보려면 `US_UNIVERSE="sp500"`,
> `MAX_TICKERS=300`, 또는 한쪽만 `MARKETS=["US"]` 로 줄이세요.
> 한국 데이터(pykrx)는 종목별 조회라 미국보다 느립니다.

### 한국투자증권(KIS) 실시간 현재가 (선택)

한국 종목의 **현재가**를 한국투자증권 Open API 실시간 시세로 보강할 수 있습니다.
(일봉·지표는 그대로 pykrx 를 사용하고, 현재가만 KIS 로 덮어씁니다.)

설정 방법:

1. 한국투자증권 계좌 개설 후 https://apiportal.koreainvestment.com 에서 개발자 등록
2. 앱키(APP KEY) · 앱시크릿(APP SECRET) 발급
3. `scan_config.py` 에 입력하고 스위치를 켭니다:

```python
USE_KIS_REALTIME = True
KIS_APP_KEY = "발급받은_앱키"
KIS_APP_SECRET = "발급받은_앱시크릿"
KIS_ENV = "real"   # 모의투자면 "vps"
```

키가 비어 있거나(기본값) 조회가 실패하면 자동으로 pykrx 값만 사용하므로 그대로 동작합니다.
추가 설치 없이 표준 라이브러리만 씁니다. (`kis_data.py`)

> 참고: 공식 종가는 미니스탁·한국투자증권·pykrx 모두 동일한 KRX 종가입니다.
> 앱마다 다른 건 장중 실시간 시세의 지연/호가뿐이라, KIS 는 '실시간 현재가' 정확도에서 이점이 있습니다.

### 추천 점수 기준 (0~5점)

상승추세(MA20>MA50) · 종가가 MA20 위 · MACD 매수권 · RSI 적정범위 · 신규 매수전환
조건을 각각 1점씩 합산하고, 골든크로스이거나 4점 이상이면 **BUY** 로 분류합니다.

## 3. 대시보드 구성 (탭 4개)

- **개요** — 전 종목의 현재가·등락·RSI·신호·전략수익률을 한 표에. 열 제목 클릭 시 정렬.
- **스크리너** — 설정 조건(RSI 범위 + 장기 이평선 위)을 통과/탈락한 종목과 사유.
- **백테스트** — 종목별 자산곡선(전략 vs 매수후보유), 수익률·CAGR·MDD·샤프·승률 등.
- **신호** — 최근 봉 기준 골든/데드크로스, RSI 과매수·과매도, MACD 전환 신호.

## 4. 종목·전략 바꾸기 — `config.py`

```python
KR_TICKERS = { "005930": "삼성전자", ... }   # 6자리 코드
US_TICKERS = { "AAPL": "Apple", ... }        # 티커 심볼

FAST_MA = 20      # 단기 이동평균
SLOW_MA = 60      # 장기 이동평균
FEE     = 0.0015  # 왕복 매매비용

SCREEN_RSI_MIN = 30   # 스크리너 RSI 하한
SCREEN_RSI_MAX = 70   # 스크리너 RSI 상한
SCREEN_ABOVE_MA = 60  # 이 이평선 위 종목만 통과
```

원하는 종목코드/티커만 넣고 다시 `python run.py` 하면 됩니다.

## 5. 파일 구조

| 파일 | 역할 |
|------|------|
| `config.py` | 종목·전략·스크리너 설정 (여기만 손대면 됨) |
| `data.py` | 시세 수집 (미국=yfinance, 한국=pykrx, 실패 시 데모) |
| `indicators.py` | SMA·EMA·RSI·MACD·볼린저밴드 |
| `screener.py` | 조건 필터링 + 최신 매매신호 |
| `backtest.py` | MA 크로스 전략 백테스트, 성과지표 |
| `dashboard.py` | 결과 → HTML 대시보드 생성 |
| `run.py` | 전체 실행 진입점 |
| `results.json` | 계산 결과 원본 (재활용·추가분석용) |

## 6. 백테스트 전략 설명

기본 내장 전략은 **이동평균 골든/데드크로스**입니다.

- `MA_fast`(단기) 가 `MA_slow`(장기) 를 **상향 돌파 → 매수**(풀포지션)
- **하향 돌파 → 매도**(전량 현금)
- look-ahead 방지를 위해 신호 발생 **다음 날 종가**에 체결, 거래 시 수수료 차감.

성과지표: 누적수익률, 매수후보유 대비, CAGR, 최대낙폭(MDD), 샤프지수, 거래횟수, 승률.

## 7. 면책

이 프로그램은 교육·연구용 도구입니다. 과거 성과가 미래 수익을 보장하지 않으며,
